Express Client
==============

Install
-------

    npm install

Start
-----

    PORT=1234 npm start

default port is 3000

Start with ` nodemon `
_(Alternatively, you can use nodemon instead of node.)_
-----

    npm run nodemon
> about nodemon: http://nodemon.io/

Test
----

    npm test
